    HSWidgetReverseDirection

    Inline Function

    HSWidgetReverseDirection

    Returns a direction that is directly opposite of specified direction.

    Declaration

    
static inline HSWidgetDirection HSWidgetReverseDirection(HSWidgetDirection direction);

    Parameters

    • direction

      direction to get opposite direction of

    Return Value

    If valid direction is passed in then a direction that is opposite of the given direction will be returned; otherwise HSWidgetDirectionNone is returned;. For instance It'll return HSWidgetDirectionRight for HSWidgetDirectionLeft, HSWidgetDirectionUp | HSWidgetDirectionLeft for HSWidgetDirectionDown | HSWidgetDirectionRight, etc.
