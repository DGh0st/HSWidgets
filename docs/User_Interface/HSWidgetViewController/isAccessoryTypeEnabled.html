    isAccessoryTypeEnabled:

    Return whether the accessory of specified type should be enabled or not.

    Declaration

    
-(BOOL)isAccessoryTypeEnabled:(AccessoryType)accessory;

    Parameters

    • accessory

      accessory type to examine
